# Fairmont Bangkok Sukhumvit — Weekly SEO Dashboard

A self-contained weekly SEO performance dashboard for internal use.

## How to use

1. Open `index.html` in any browser — no server needed
2. Tick off tasks in the action plan as the team completes them
3. Click **Refresh** to update the timestamp

## Live URL (GitHub Pages)

Once deployed: `https://YOUR-USERNAME.github.io/fairmont-bangkok-digital-marketing-hub`

## Update every Monday

1. Ask Claude to regenerate the dashboard with new data
2. Download the new `index.html`
3. Go to this repo → click `index.html` → pencil icon → paste new content → Commit

## Data sources

- Google Search Console (manual check)
- PageSpeed Insights
- Manual SERP checks (TH/SG IP)
- Third-party keyword tools (Ubersuggest / Google Keyword Planner)

> **Note:** GA4 access is restricted by corporate. Dashboard will be updated once access is granted.
